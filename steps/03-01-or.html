You can match different words with different content. Many programming languages use the character "|" as a replacement for "or".

In the following list, you can filter all villages which contain hausen or roda by writing hausen|roda. I read hausen|roda like "hausen or roda". To write the character |, you can use the key next to the return key on the right side of the keyboard.
