Before, you could describe the set of all digits using [0-9]. Because some sets are used regularly, there are short notations for them.

  • \d is the set of add digits.
  • \s is the set of all spaces, e.g. or the line
    break.
  • . matches all characters except the line break. .* can be used if you do not care about some content.
  • \w is for letters in words. \w is the same as [0-9a-zA-Z]_.
  • \\ matches the backslash \.
Additionally, if you want to invert a set like \s, you can use the capital letter e.g. \S.

The following text is from "Die Zwölf Weihnachtsfeiertage" by Gerhard Schone [source]. Here, animals are counted. Find all lines in which a number is followed by space followed by a word.

  • Am 5. Weihnachtstag
  • schenkte mir mein lieber Schatz
  • 5 Füllhörner
  • 4 Pfirsiche
  • 3 Drosselchen
  • 2 Zwiebelchen
  • und für’n Birnenbaum 1 Spatz.
